# React Weather Application
openweathermap 날씨 API를 이용하여 지역의 날짜, 날씨, 그리고 온도를 보여주는 서비스입니다.

## Tech Stack
* React.js

## Feature
* 입력 폼에 검색을 원하는 지역을 영어로 입력하여 해당 지역에 대한 날씨 정보를 화면에 보여줍니다.
* 날씨에 따라 다른 배경사진을 보여주어 시각적인 효과를 주었습니다.
